# ReleaseSeal 1.0.0 (14)

Initial public release of ReleaseSeal for macOS 12 or later.

ReleaseSeal provides an offline SwiftUI app and bundled command-line tool for layered inspection of DMG, PKG, ZIP, and app artifacts. It compares signature identities, exact artifact hashes, tracked files, and compromised evidence with a reviewed public database. Optional deep scanning reports internal signer details when more context is useful.

ReleaseSeal does not execute inspected software and does not claim that a matching artifact is malware-free.
